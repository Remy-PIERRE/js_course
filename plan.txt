JS

1 - Intro

a - context
=> technologie incontournable du web
=> utilisé en front pour créer des pages dynamiques (vanilla / react / angular / vue.js)
=> utilisé en back pour serveur (Node.js)

=> attention java !== javascript

b - caractéristiques
=> langage interprété (pas de compilation)
=> typage faible
=> asynchrone 
=> basé sur la gestion d'évenements (focus sur interactions utilisateur)
=> basé objet (peu intégrer les principes de la POO)


2 - Syntaxe de base

a - Variables
=> let 
=> const
=> var !!!

=> protée (scope) et blocs de code

b - Structures de contrôle
=> conditions (if, else, else if, switch, ternaire)
=> boucles (for, while, do while)
=> mots cléfs (break, continue)

c - Fonctions
=> déclaration
=> arguments
=> valeur de retour
=> fonctions fléchées
=> fonctions anonymes
